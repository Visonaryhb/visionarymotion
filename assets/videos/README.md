# Video Hosting für Visionary Motion

## Lokale Videos für eingebettete Wiedergabe

Platzieren Sie Ihre MP4-Videos in diesem Ordner für echte eingebettete Wiedergabe ohne externe Weiterleitungen.

### Empfohlene Videoformate:
- **Format**: MP4 (H.264 Codec)
- **Auflösung**: 1920x1080 (Full HD)
- **Bitrate**: 5-10 Mbps für hohe Qualität
- **Audio**: AAC Codec

### Dateibenennung:
- `automotive-showcase.mp4`
- `event-highlights.mp4` 
- `gastronomie-presentation.mp4`
- `retail-commercial.mp4`
- `hochzeit-romantic.mp4`

### Verwendung im HTML:
```html
<button class="video-card" 
        data-category="Automotive" 
        data-video="assets/videos/automotive-showcase.mp4"
        data-thumb="assets/videos/automotive-showcase-thumb.jpg">
```

### Vorteile lokaler Videos:
✅ Vollständige Kontrolle über Wiedergabe
✅ Keine externen Abhängigkeiten  
✅ Kein Verlassen der Website
✅ Bessere Performance
✅ Konsistente Benutzererfahrung
